# **App Name**: DataConnect Nigeria

## Core Features:

- Network Selection: Allow users to select their network provider from a list (MTN, Airtel, Glo, 9mobile).
- Data Plan Display: Display available data plan options with their respective prices.
- User Information Form: Collect user's phone number, name, and optional email/referral code.
- WhatsApp Order Submission: Prefill a WhatsApp message with order details and open WhatsApp Web/API with the message sent to +2349040367103.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to reflect connectivity and trustworthiness.
- Background color: Light gray (#F0F2F5) for a clean and modern look.
- Accent color: Green (#228B22) to highlight key actions like 'Buy Now'.
- Body and headline font: 'PT Sans' (sans-serif) for easy readability and a modern feel.
- Mobile-first, responsive design to ensure optimal viewing experience across devices.
- Use recognizable icons for network providers and data amounts, sourced from a free icon set (e.g., FontAwesome or Material Icons).
- Subtle transitions and animations for form input and button clicks to enhance user experience.
